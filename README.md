## Installation

## Running Server

```bash
  git clone https://github.com/rohanshrestha09/Patra.git
  cd Patra
  npm install
  npx tsc --watch
  npm run start dist/server
```

## Running Client

```bash
  git clone https://github.com/rohanshrestha09/Patra.git
  cd Patra/client
  npm install
  npm run start
```
